## 학습, 새로배운 내용

- Django를 이용해 홈페이지, 각 영화의 정보, 글작성, 수정, 로그인 등 이번 한주간 학습했던 모든 내용을 복습하는 시간을 가졌다.
- bootstrap_form을 이용하여 Django가 제공해주는 form을 더 보기 좋게 바꾸는 방법을 학습하였다.

## 어려웠던 부분

- Django에서 오류가 발생했을 때 어디서 발생한 오류인지 찾아내는 것
